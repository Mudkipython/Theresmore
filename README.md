## 新手必读

### 第一次玩选什么难度？

**建议选普通难度，不要选简单难度。**

原因：

- 简单和普通的敌人强度没有区别。
- 简单难度不能获得许多成就。
- 少拿成就会导致后续发展到第五时代附近时更容易卡死。

如果刚开局不久且选了简单，建议立刻在右下角切换为普通难度。

### 第一次玩路线建议

第一次玩建议：

1. **少打支线，优先走主线。**
2. 支线收益通常偏低，但敌人偏难。
3. 过早推进某些战争线会导致反复被入侵，体验很差。
4. 关键节点前务必手动存档，尤其是：
   - 月明之夜
   - 亡者之门
   - 兽人线
   - 高骷髅 Boss 战

## 常见卡关点

| 阶段 | 卡点 | 解决方式 |
|---|---|---|
| 第二时代 | 解锁守望者前哨后没后续 | 建造 4 个守望者前哨，解锁月明之夜 |
| 第二时代 | 打赢月明之夜后卡住 | 把几个主要蓝色建筑升到 3 级，具体看日志提示 |
| 第四时代 | 堕天使后卡住 | 建造港湾区，探索并击败远西岛，继续第五时代相关研究 |
| 第五时代 | 兽人线卡住 | 打赢所有兽人堡垒，再打赢兽人部落，之后研究并建造治疗圣所 |
| 第六时代 | 深渊主线卡住 | 主要流程需要打赢失落山谷、腐化之地、黑暗村庄 |

> 如果研究全点完但没有新内容，通常不是 bug，而是需要：  
> **建造指定建筑 → 探索 → 战斗 → 解锁新研究**。

## 幸运石与幸运建筑

幸运石用于购买部分幸运建筑。数量显示在“人类”面板中。

当前版本幸运石获取上限约为 **250 个**，主要通过灭世获得：每次灭世 +2 幸运石。

### 推荐优先级

幸运石不足时优先：

1. **炮兵军官营房 10 级**
2. **佣兵营地 10 级**
3. **地下隧道 15 级**
4. 卡研究上限时考虑 **15 级灵感大厅** 解锁的考古挖掘场
5. 富余时先点：
   - 10 级灵感大厅
   - 10 级娱乐区域

## 侦察兵区别

| 单位 | 死亡概率 | 找国家 | 找敌人 | 找资源 | 无事发生 |
|---|---:|---:|---:|---:|---:|
| 侦察兵 / 无人机 | 10% | 20% | 20% | 20% | 30% |
| 探险家 | 1/12 | 2/12 | 5/12 | 2/12 | 2/12 |
| 使魔 | 0 | 0 | 2/6 | 3/6 | 1/6 |

备注：

- 无人机需要打败科布自治领后获得。
- 使魔无法探索国家，但不会死亡，而且能带回资源；适合挂机。
- 如果缺的是国家，仍然需要侦察兵或探险家。
- 如果长时间不出新敌人，通常不是侦察兵数量问题，而是缺前置研究/战斗/时代推进。

## 战斗机制

> 下文中“冲击-战士-1 个战士”对应“类型-兵种-士兵”。

### 克制关系

```text
肉盾克骑兵
骑兵克远程
远程克冲击
冲击克肉盾
```

也可以记为：

```text
盾 ← 冲 ← 弓 ← 骑
右边克左边，肉盾克骑兵
```

### 每一轮战斗流程

1. 我方按进攻顺序逐个士兵造成伤害。
2. 敌方按防守顺序逐个士兵承受伤害。
3. 若我方士兵克制敌方士兵，本次伤害 ×2。
4. 若承受伤害小于血量，累计伤害。
5. 若承受伤害大于血量，该敌方士兵进入濒死状态。
6. 一般伤害不溢出；有践踏伤害的单位会把溢出伤害按比例继续攻击下一个敌人。
7. 濒死单位本轮仍可攻击。
8. 双方完成攻击后，濒死士兵死亡。
9. 如果双方都还有士兵，进入下一轮。
10. 如果双方同归于尽，判玩家失败。

### 新版战斗机制重点

新版战斗和旧版最大区别：

- 数量非常重要。
- 有范围伤害和践踏伤害的单位价值上升。
- 基地防御很有用，基地会优先于守城部队承伤。
- 克制只在士兵个体攻击时单独计算，不再是整队全程翻倍。
- 不用过度担心“多带一个兵种导致必输”的旧版反直觉情况。
- 战斗前仍建议存档，计算器/模拟器只能作为参考。

## 配兵思路

核心原则：

> **数量为王。狮子搏兔，亦用全力。**

### 通用配队思路

1. 选择能一击必杀敌方主力，且有范围伤害的兵种作为主攻。
2. 用高血量单位承伤。
3. 想减少主力损耗，就堆便宜肉盾。
4. 多数时候不要只迷信远程先手，因为濒死机制下敌人本轮仍能反击。
5. 大多数普通战斗可以用主冲击兵种平推。
6. 高难战斗前务必用新版模拟器估算，并存档。

新版战斗模拟器：

```text
https://theresmoregamebc.g8hh.com.cn/
```

> 注意：模拟器部分克制机制可能仍有旧版本差异。若模拟器显示能打但游戏内失败，以游戏实测为准，并根据敌人克制关系增加兵力。

## 入侵 / 防守型敌人

这些敌人会主动进攻。防守失败通常导致资源清零，部分事件可能强制转生。

| 敌人/事件 | 触发条件 |
|---|---|
| 狗头人王国 | 研究狗头人的国家 |
| 野蛮人部族 | 研究野蛮人部落 |
| 窃魂者堡垒 | 研究巨大洞穴 |
| 兽人突击队系列 | 研究兽人的威胁 |
| 兽人全家 | 12 级定居点研究“被烧毁的农场”，直到打赢兽人部落 |
| 莫德凯男爵 | 5 个光之广场 |
| 恶孽之手 | 研究激活信号机器 |
| 炎骑士城堡 | 打赢黑暗骑士巡逻队后研究探索出 |
| 烬火灾主 | 研究荒漠的精灵神话，打南火荒漠、火法师领域入口后探索出 |
| 织田军、青龙 | 和恩索诸众结盟后选择和织田开战 |
| 中立国家 | 被玩家宣战后的 8 个中立国家 |
| 亡者之门 / 无智之恶防守战 | 防守失败会强制转生 |

### 应对建议

如果暂时打不过持续骚扰：

```text
驻军 = 0
主力全部放进攻击队列
资源被抢就被抢
不要白白死兵
```

这对野蛮人/兽人骚扰尤其重要。

## 军队上限来源

零转也能获得大量军队上限，兽人前理论上可做到约 950+ 上限，足够打赢兽人。

常见来源：

- 军事祖先：+25 军队上限
- 阿塔玛雕像：+15 军队上限
- 各种军事建筑
- 专注魔法的魔法马厩：+100 军队上限
- 与国家结盟：联盟大使馆提供 150+ 军队上限
- 值守仓库 / 大型仓库 / 物流中心：提供 100+ 军队上限
- 传承“匹夫有责”：后期可提供 2000+ 军队上限
- 灭世终焉转生：每次永久 +10 军队上限

后期军队数量参考公式：

```text
N = 1350 * lg(超转上限) + 4500 + 10 * 灭世数
```

其中：

- 加号前半部分：无上限建筑提供的军队上限
- 加号后半部分：有上限建筑、传承、灭世等提供的上限

## 结盟与征服

### 结盟

- 给固定值产量。
- 给少量军队上限。
- 前四个国家结盟可解锁联盟大使馆。
- 联盟大使馆每个提供：
  - +8 军队上限
  - +2/s 研究
  - +2/s 黄金

### 征服

- 给百分比产量，通常略高于结盟固定值。
- 前四个国家、科布统治领和恩索诸众有专属支线。
- 其余国家通常直接给声誉。

### 推荐

- 前四个国家推荐结盟，大使馆提供的军队上限很关键。
- 如果确认本轮战斗压力不大，可以抽空征服一次拿成就。
- 夜谷保护国征服给不错资源加成。
- 恩索诸众结盟线收益明显高于征服，但结盟线战斗更难。
- 其他国家一般推荐征服。

## 难度区别

| 难度 | 敌人数量倍率 | 说明 |
|---|---:|---|
| 简单 | 1 倍 | 可预测战斗结果；战死士兵部分改为受伤；无法获得建筑数量成就 |
| 普通 | 1 倍 | 推荐首次游玩；可预测战斗结果 |
| 困难 | 1.5 ~ 1.65 倍 | 部分野怪提供成就 |
| 无望 | 2 ~ 2.4 倍 | 更高敌人数 |
| 神明 | 4.5 ~ 5.85 倍 | 可建造某建筑提高声誉产出 |

备注：

- 简单和普通敌人强度一样，但简单会损失成就，首次游玩不推荐。
- 高难度下获得成就时，如果低难度同名成就未获得，也会同时获得。
- 部分 Boss 型敌人数量不随难度变化，例如泰坦本体。

- 2000 信念上限
- 3 级魔法环

推荐：

- 前中期：**研究路线**
- 后期：**魔法路线**

原因：

- 零转信仰上限不足时，只有研究路线更容易到 13 级魔法环。
- 研究路线提供大量法力、水晶、研究上限，减少卡时代。
- 魔法路线后期通过魔法马厩提供军队上限，神圣魔像血量加成很强。

### 9. 军事定居点 / 信仰灯塔 / 生产中心

每条线都有奇观成就，最终都要走一次。

## 转生 / Prestige

### 第一次什么时候转生？

推荐标准不是固定时间，而是：

> 玩到进度明显变慢、继续推进很痛苦时再转生。

首次转生提升感有限，不会转一次立刻起飞。

常见第一次转生窗口：

- 需要建造法力深井但上限卡得很厉害
- 纳红石时代建好光之城后
- 兽人线开始需要长时间屯兵且骚扰影响挂机
- 已经完成多个 Prestige 入口，继续打收益很低

### 各种转生方式区别

| 转生方式 | 奖励 |
|---|---|
| 飞升 | 智慧卷轴 |
| 富甲天下 | 代币 |
| 亡者之门 | 遗物 |
| 定居点奇观 | 宝石 |
| 泰坦 | 泰坦的礼物 |

重要机制：

> 每个转生方式完成之后，无论最后点击哪个转生按钮，都可以获得所有已完成转生方式的奖励。

## 传承 / Legacy 推荐

### 最能改变体验的传承

#### 1. 工匠公会

极高优先级。

- 提供原料、钢、补给、水晶等高级资源固定产出。
- 这些资源本来要占人口和低级资源生产。
- 从第二时代到第五时代体感明显。
- 点数消耗不算高。

#### 2. 泰坦的建筑

极强，但需要打败泰坦。

- 提供大量木材、石头、工具、铜、铁产出和上限。
- 几乎解放木材和石头压力。
- 点数消耗很高，但非常值得。

#### 3. 泽尼克斯的使魔

便宜且好用。

- 只需 4 点。
- 解锁使魔侦察兵。
- 不会死，不消耗黄金，能带回资源并提供研究。
- 缺点：不能探索国家。

#### 4. 内政部

宝石首选之一。

- 大量研究点。
- 中量黄金、补给。
- 大量相关资源上限。
- 额外教授间接提供水晶。
- 需要上周目完成第五时代奇观并获得宝石。

### 前期常见高价值传承

| 传承 | 评价 |
|---|---|
| 工匠公会 | S 级，优先点 |
| 内政部 | 宝石首选 |
| 泰坦的建筑 | 泰坦礼物首选 |
| 店主和饲养员 | 工具、牛、马，长期有用 |
| 工匠 | 二级资源增益，长期强 |
| 强化斧头 / 灌溉 / 著名石匠 | 基础资源加成，早中期好用 |
| 区域市场 | 代币首选之一，黄金和上限 |
| 资源上限 | 早期休闲玩法重要，和超转上限协同 |
| 战争部 / 宗教部 | 宝石可选，简化军队/魔法 |
| 聪明的村民 | 研究不足时很关键 |
| 登攀者世界图书馆 | 便宜且前期全面增益 |
| 修道会 | 研究、食物、信念、法力都给 |
| 强化信念 | 信仰/法力相关路线好用 |
| 战斗天使 | 有人喜欢，但前期信仰法力紧张，后期会被更强单位超越 |

### 关于战斗天使

战斗天使不是垃圾，但不一定是第一优先。

优点：

- 第二轮前中期战斗体验较好。
- 不完全依赖普通军队资源体系。
- 乐土线和后续建筑能继续强化。

缺点：

- 需要智慧卷轴。
- 前期信仰和法力还有很多其他用途。
- 后期会被更高阶单位超越。

如果目标是快速重回兽人线，优先级通常低于：

1. 工匠公会
2. 内政部
3. 智慧卷轴相关强力建筑
4. 粮食/工具/研究/资源上限类传承

## 速转、超转、灭世

### 什么是速转？

速转指：

```text
开局直奔月明之夜及前置研究
不造兵
直接送死重置
快速获取传承点与转生加成
```

长线发展中，速转效率很高，但会牺牲正常游玩体验。

### 速转建议

只推荐在：

- 已了解流程
- 想刷转生加成
- 能接受重复流程

时使用。

速转中额外做事的优先级：

1. 必经研究中顺手给声誉的研究，如战争、公会
2. 5/10/15 个生产建筑研究
3. 建栅栏和墙：不推荐，读条和资源成本高
4. 打月明：不推荐，时间成本不回本
5. 出兵打架：不推荐

### 超转生效果

截至 1.2.2：

1. 每拥有一个传承，超转时提高：
   - 0.1% 全局产量
   - 1% 全局上限
2. 超转会让探索和战斗耗时减半，最多 5 次达到最低：
   - 初始 60 秒
   - 最低 3 秒
3. 建筑耗时最多 4 次超转达到最低：
   - 初始 1.4 秒
   - 最低 0.1 秒
4. 第一次超转解锁强力前期建筑。
5. 第一次超转解锁强力前期法术。
6. 第二次超转解锁肉盾兵种和相关建筑。
7. 第三至五次超转解锁三个科技系列 I/II/III，为低档兵种提供大量加成。

### 灭世效果

每次以灭世终焉方式转生：

- +4% 资源产量
- +2% 声誉加成
- +2 光明
- +2 幸运石
- +10 军队上限

这些加成永久保留，超转后仍有效。

### 超转还是灭世？

简单结论：

- **5 超前：不建议灭世。**
- **5~80 超：不建议大量灭世。**
- **80 超后：根据需求选择连续大量灭世或继续超转。**

核心原因：

- 超转更快提高资源上限与行动速度。
- 灭世需要深入单局后期，前期效率低。
- 灭世要连续大量做才有质变，不适合边超边灭。

### 为什么不推荐边超边灭？

因为边超边灭会舍弃资源加成中的传承和转生乘区，本质上是降低灭世速度去贪一点超转加成，通常不如连续灭世。

> 以下内容是旧版本归档，不适用于 1.2.2 的新版战斗系统。

- 进攻顺序：弓 > 冲 > 盾 > 骑

- 战斗曾经更接近“总攻击 vs 总防御”的比大小算法
